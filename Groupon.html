Class Method Details

+ (Array<Hashie::Mash>) deals(options = {})

Returns a list of deals. The API returns an ordered list of deals currently running for a given Division. Priority is based on position within the response (top deals are higher in priority).

Parameters:

  • (Hash) options (defaults to: {}) —

    a customizable set of options

Options Hash (options):

  • (String) :lat — default: Latitudinal coordinates based on IP of API request —

    Latitude of geolocation to find deals

  • (String) :lng — default: Longtitudinal coordinates based on IP of API request —

    Longitude of geolocation to find deals

Returns:

  • (Array<Hashie::Mash>) —

    an array of deals
